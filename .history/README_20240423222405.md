## How to build

> Only works on Windows and Linux! Mac systems not supported!

```ps
git clone https://github.com/Sherry65-code/EclipseEngine --depth 1 --recursive
cd EclipseEngine
mkdir build
cd build
cmake ..
.\EclipseEngine.sln
```
